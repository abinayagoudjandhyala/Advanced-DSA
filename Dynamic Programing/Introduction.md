

# 📘 DP – Day 1 (Introduction)

## 🚀 What I Learned

* Dynamic Programming is used to **avoid repeating the same calculations**
* It works when problems have:

  * overlapping subproblems
  * optimal substructure

---

## 🔁 Memoization (Top-Down)

* Uses **recursion + storing results**
* If a value is already calculated → reuse it
* Saves time by avoiding repeated calls

👉 Simple idea:

> "Don’t calculate again if already solved"

---

## 📊 Tabulation (Bottom-Up)

* Uses **loops instead of recursion**
* Start from base case and build up
* More efficient (no recursion stack)

👉 Simple idea:

> "Build answer step by step"

---

## ⚡ Difference (My Understanding)

* Memoization → recursive, easy to write
* Tabulation → iterative, better optimized

---

## 🧠 My Key Takeaway

DP =
👉 Break problem into smaller parts
👉 Store results
👉 Reuse them
